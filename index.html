Yapay Zeka ile
Güvenlik Zaafiyeti Tespiti

Yazılım kodlarındaki güvenlik zaafiyetlerini otomatik olarak tespit eden gelişmiş yapay zeka teknolojisi ile yazılım geliştirme süreçlerinizi daha güvenli hale getirin.

Proje Amacı

Gençtek AI olarak geliştirdiğimiz çığır açan teknoloji

Gençtek AI olarak, yazılım kodlarındaki güvenlik zaafiyetlerini otomatik olarak tespit edebilen bir yapay zeka (YZ) modeli geliştirmeyi hedefliyoruz.

Geleneksel güvenlik denetimi yöntemleri genellikle zaman alıcı ve insan hatasına açık olabilirken, geliştireceğimiz bu yapay zeka, potansiyel güvenlik açıklarını daha hızlı ve verimli bir şekilde belirleyerek yazılım geliştirme süreçlerini daha güvenli hale getirecektir.

Veri Analizi

Makine Öğrenimi

Güvenli ve zaafiyetli kod kalıplarını öğrenir ve sınıflandırır

Tespit

Yeni kod parçacıklarında güvenlik açıklarını otomatik olarak tespit eder

Bu süreç, bir doktorun hastalıkları teşhis etmesi gibidir: Doktor (YZ modeli) binlerce hasta vakasını (kod örneğini) inceleyerek hastalık semptomlarını (zaafiyet kalıplarını) öğrenir ve ardından yeni bir hastanın semptomlarına bakarak hastalığı teşhis eder.

Sistemimizin nasıl çalıştığını görün

Zaafiyetli Kod

username = request.args.get('username')
query = "SELECT * FROM users WHERE username = '%s'" % username
cursor.execute(query)
SQL Injection

Kullanıcı girdisi SQL sorgusuna doğrudan eklenmiş. Bu, SQL injection saldırılarına açıktır.

Güvenli Kod

username = request.args.get('username')
cursor.execute("SELECT * FROM users WHERE username = %s", (username,))
Güvenli

Parametreli sorgu kullanılarak SQL Injection önlenmiş.

Zaafiyetli Kod

@app.route("/search")
def search():
    q = request.args.get("q")
    return f"<h1>{q}</h1>"
Cross-Site Scripting (XSS)

Kullanıcı girdisi HTML çıktısına doğrudan eklenmiş. XSS saldırılarına açıktır.

Güvenli Kod

@app.route("/search")
def search():
    q = request.args.get("q")
    return render_template("results.html", query=q)
Güvenli

Jinja2 template otomatik HTML escape özelliği sayesinde güvenlidir.

Zaafiyetli Kod

import os
command = request.args.get('cmd')
os.system(command)
Command Injection

Kullanıcı girdisi kabuk komutuna doğrudan aktarılmış. Sistem komutları çalıştırılabilir.

Güvenli Kod

import subprocess
command = ["ls", "-la"]
result = subprocess.run(command, capture_output=True, text=True)
Güvenli

Komutlar liste şeklinde verilmiş, shell=True kullanılmamış.
